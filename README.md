## 🚀 快速开始

### 1. 安装 Rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. 克隆仓库

```bash
git clone <repository-url>
cd CCRustStudy
```

### 3. 开始学习

从第一个模块开始：

```bash
cd 01-foundation/01-ownership-basics
cat TUTORIAL.md  # 阅读详细教程
cargo run        # 运行示例代码
cargo test       # 运行测试
```

### 4. 选择学习路径

根据你的目标选择合适的学习路径：

- **Web 开发** → 完成基础篇、中级篇、Web 服务、AI Gateway
- **系统编程** → 完成基础篇、中级篇、高级篇
- **图形编程** → 完成基础篇、部分中级篇、图形基础、3D 渲染器

## 🔧 常用命令

```bash
# 构建和运行
cargo build              # 构建项目
cargo build --release    # 优化构建
cargo run                # 运行项目
cargo run -- <args>      # 带参数运行

# 测试
cargo test               # 运行所有测试
cargo test <test_name>   # 运行指定测试
cargo test -- --nocapture # 显示 println! 输出

# 代码质量
cargo check              # 快速检查编译错误

cargo fmt                # 格式化代码
cargo doc --open         # 生成并打开文档
```
